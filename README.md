# KuaiSearch Demo Data

This repository contains **demo data of KuaiSearch**, released **for demonstration purposes only**.

The demo data is intended to illustrate:
- the data format and schema,
- the organization of files,
- the overall structure of the KuaiSearch dataset.

⚠️ **Important Notice**  
The **full KuaiSearch dataset** and **complete baseline implementations** will be **publicly released after the review process is completed**.

This demo version does **not** reflect the full scale or data distribution of the final dataset and should **not** be used for quantitative evaluation or benchmarking.
